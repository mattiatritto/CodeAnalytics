- Install gcloud cli;
- Create a new project inside Google Cloud Platform;
- Search for "Artifact Registry API", "Cloud Build", "Cloud Deploy" and enable these;
- gcloud auth login
- gcloud config set project software-cost-estimation
- gcloud artifacts repositories create software-cost-estimation --repository-format=docker --location=us-central1 --description=“Docker backend” 
- gcloud builds submit --region=us-central1 --tag us-central1-docker.pkg.dev/software-cost-estimation/software-cost-estimation/software-cost-estimation-image:tag1
- gcloud run deploy --image=us-central1-docker.pkg.dev/software-cost-estimation/software-cost-estimation/software-cost-estimation-image:tag1 
- Check if it is running at: https://software-cost-estimation-image-5699838011.us-central1.run.app/docs
